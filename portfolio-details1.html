CrashVision Project Details

Project information

  • Category: ASP .NET Core MVC Web App
  • Project date: 15 February, 2023
  • Project URL: www.github.com/PrathamSavjani/CrashVision-Road-Safety

Our team participated in The Gateway Corp's 30-hour .NET Hackathon to create a Road Safety app. Developed a website to display accident data on OpenStreetMap with weather info using .NET MVC 6.0, JavaScript, RapidAPI, and MS SQL (2018).
