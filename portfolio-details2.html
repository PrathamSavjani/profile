Mag Stack Project Details

Project information

  • Category: E-Commerce web app
  • Project date: 06 April, 2023
  • Project URL: www.github.com/PrathamSavjani/MagStack

Developed e-commerce website for magazine using ASP.NET Core MVC 6.0 with Entity Framework. Implemented N tier architecture & repository pattern for scalability and maintainability. Implemented role-based toggling of functions. Utilized MS SQL (2018) for DB integration.
